# DotFiles

Used on MacBookPro 10.9 & 10.11

```bash
./install.sh
./deps.sh
```

* php70-xdebug

* wget

* iperf
* google-chrome
* vlc
* vagrant
* virtualbox
* spectacle
* sublime
* slack
* skype
* phpstorm
* limechat
* iterm
* firefox
* filezilla
* alfred2
* git-crypt
